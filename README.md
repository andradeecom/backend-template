# Backend Template

This is a backend template with NestJS and TypeScript.

## Features

- [x] NestJS
- [x] TypeScript
- [x] ESLint
- [x] Prettier
- [x] Husky
- [x] Lint-staged
- [x] Jest
- [x] Supertest
- [x] JWT Strategy
- [x] Swagger Documentation
- [x] Prisma ORM
- [x] Resend (Email Service)
- [ ] Login with Google (Social Login)
- [ ] Login with Apple (Social Login)
